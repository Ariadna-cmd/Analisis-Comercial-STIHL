# Análisis Comercial y Detección de Riesgo de Fraude — STIHL

## Descripción
Análisis de desempeño comercial de productos STIHL a partir de 
datos reales de ventas. El proyecto busca transformar registros 
comerciales en indicadores estratégicos e identificar patrones 
anómalos que puedan representar riesgo de fraude.

## Objetivos
- Analizar el comportamiento de ventas por producto, categoría, 
  cliente y zona geográfica
- Generar métricas de desempeño comercial para soporte de 
  decisiones estratégicas
- Detectar patrones anómalos en transacciones como señal de 
  riesgo de fraude

## Tecnologías utilizadas
- SQL
- Power BI (dashboard de visualización)
- Excel

## Estructura del proyecto
- `/data` — datasets utilizados
- `/dashboard` — capturas del dashboard en Power BI

## Resultados principales
- Identificación de productos y categorías con mayor volumen 
  de ventas
- Segmentación de clientes por comportamiento de compra
- Detección de transacciones con patrones atípicos
- Dashboard interactivo con KPIs comerciales clave

## Contexto
